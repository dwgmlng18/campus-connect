# ATURAN WAJIB SEBELUM MENGERJAKAN TASK

## 1. Scan folder skill
Sebelum menulis, mengedit, atau menjalankan kode apapun, buka dan baca
seluruh isi folder `.agent/skills`, termasuk semua subfolder:
- agentic-awesome-skills
- claude-seo
- skillx
- ui-ux-pro-max-skill

## 2. Baca satu per satu
Buka file utama di tiap subfolder (SKILL.md, README.md, atau index-nya).
Baca berurutan, jangan ada yang dilewati, meskipun terlihat tidak relevan
di awal — relevansi baru bisa dipastikan setelah isi dibaca.

## 3. Ringkas sebelum lanjut
Setelah membaca tiap skill, buat ringkasan singkat (2-4 poin) berisi:
- Fungsi/tujuan skill tersebut
- Aturan atau konvensi penting yang harus diikuti
- Kapan skill ini relevan dipakai

## 4. Petakan ke task saat ini
Setelah semua skill terbaca, tentukan skill mana yang relevan dengan
task yang diminta, lalu jelaskan singkat kenapa skill itu dipilih
sebelum mulai eksekusi.

## 5. Ikuti aturan project lain
Jika ada dokumentasi tambahan (misal breaking changes Next.js di atas,
atau file lain di project) yang menyebutkan konvensi khusus, baca juga
dan patuhi — jangan asumsikan berdasarkan pengetahuan umum/training data.

## 6. Baru mulai kerja
Setelah langkah 1-5 selesai dan dikonfirmasi, baru mulai menulis/mengedit kode.
Jika di tengah pengerjaan ditemukan skill baru yang relevan, hentikan
sejenak, baca skill tersebut, lalu lanjutkan.

## Catatan
Tujuan aturan ini: memastikan setiap keputusan teknis (struktur kode,
gaya penulisan, SEO, UI/UX) konsisten dengan skill yang sudah didefinisikan
di project, bukan berdasarkan asumsi umum.
